# Baby Bard Games — Portfolio Site

The solo-dev studio of Daniel Lakutis.
Static site, no build step, no external dependencies.

- `index.html` — studio home: all projects, dropdown nav, per-project sections
- `velflora.html` — the Velflora flagship showcase (world, Bastion, Callings, combat, books, the video game)
- `bastion-map.html` — interactive Bastion city map (player-safe surface layer)
- `assets/` — all images, web-optimized
- `styles.css` — shared stylesheet

**View it:** double-click `index.html`, or serve the folder
(`python -m http.server`) for cleaner URLs.

**Host it:** upload the folder as-is to any static host
(GitHub Pages, Netlify, itch.io page, etc.). Everything is relative paths.
